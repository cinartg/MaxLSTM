# MaxLSTM
I am trying to create an LSTM that is "maxed-out"; that is, it is meant to be the most *POWERFUL* type of the standard LSTM.

The LSTM, or Long Short-Term Memory is an upgrade from the standard Recurrent Neural Network (or "RNN") architecture. It implements dedicated memory cells and gates that allow the  network to "remember" or forget information over longer periods of time.

## DETAILS ##

- Language: *Python 3.14.6* 
- GPU Used: (x2) T4
